Desarrollo de un Sistema de Control de Registro de Pacientes

Este proyecto busca mejorar la gestión del área de Psicología del Centro de Salud de Chongoyape, optimizando el control de pacientes, evitando duplicidades y facilitando el seguimiento terapéutico.

Objetivos del Proyecto

  • Implementar un sistema digital de registro de pacientes.
  • Centralizar la información en una base de datos organizada.
  • Facilitar el seguimiento terapéutico y la generación de reportes.
  • Optimizar la atención psicológica y la toma de decisiones.
